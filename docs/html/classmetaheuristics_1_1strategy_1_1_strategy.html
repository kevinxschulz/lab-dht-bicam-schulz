Detailed Description

Central strategy controller for executing metaheuristic experiments.

This singleton coordinates generators, execution flow, stopping criteria, statistics and bookkeeping for a metaheuristic run. It builds and switches generators, stores the best solution found and manages timing and logging options used across the framework.
