\doxysubsection{Detailed Description}
Abstract base class for solution evaluation methods used by the problem extension mechanisms. Concrete subclasses implement a strategy for evaluating a {\ttfamily \doxylink{classproblem_1_1definition_1_1_state}{problem.\+definition.\+State}} and storing the resulting evaluation vector in the provided state.
