\doxysubsection{Detailed Description}
Multi-\/objective stochastic hill climbing generator.

Explores the neighborhood of a reference state and chooses candidate solutions using a multi-\/objective acceptance rule. Intended for problems with multiple objectives where nondominated moves are preferred.
