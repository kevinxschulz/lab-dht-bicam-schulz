Evaluate the given state for each objective function. 

The method takes into account the overall problem type (maximize / minimize) and each objective\textquotesingle{}s declared type. If necessary, it inverts the objective value so that the returned evaluations are consistent with the problem direction (higher = better when the problem is a maximization problem).
