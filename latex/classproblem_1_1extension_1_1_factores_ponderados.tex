\doxysubsection{Detailed Description}
Weighted-\/factors (scalarization) solution method.

This evaluator computes a single scalar value by combining each objective\textquotesingle{}s (possibly inverted) evaluation with its configured weight. The computed scalar is stored as a single-\/element evaluation vector in the provided {\ttfamily \doxylink{classproblem_1_1definition_1_1_state}{State}}.
